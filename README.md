#  Analyse et prédiction de la performance académique des étudiants

##  Description du projet
Ce projet vise à analyser les facteurs qui influencent la performance scolaire des étudiants à partir du jeu de données *Student Academic Performance Dataset*.
L’objectif est d’identifier les éléments académiques, sociaux et personnels ayant le plus d’impact sur les résultats finaux, et de construire des modèles de prédiction fiables.

## Description du jeu de données
- Taille : 6 607 enregistrements  
- Variables : 20 colonnes (7 quantitatives et 13 qualitatives)  
- Variable cible : `Exam_Score`  
- Exemples : `Hours_Studied`, `Attendance`, `Sleep_Hours`, `Previous_Scores`, `Motivation_Level`, `Family_Income`

##️ Méthodes envisagées
- Classification non supervisée : regroupement des étudiants selon leurs profils.
- Classification supervisée : prédiction du niveau de réussite (faible / moyen / élevé).
- Régression linéaire : modélisation du score selon les facteurs.
- Régression non linéaire : amélioration de la précision via Random Forest ou XGBoost.
